Object Model
html에서의 태그는 브라우저가 알아서 객체화 시켜 준다. 따라서 javascript로 접근하여 메소드 등을 실행 할 수 있게 되는 것이다.
즉, 우리는 객체화된 태그를 찾아서 동적으로 웹사이트에 영향을 줄 수 있다.

window객체에 속해있는 것으론 DOM, BOM, Javascript 가 있는데
다시 DOM의 객체로는 document
     BOM의 객체로는 navigator, screen, location, frames, history, XMLHttpRequest
     Javascript로는 Object, Array, Function ... 등이 있다.
